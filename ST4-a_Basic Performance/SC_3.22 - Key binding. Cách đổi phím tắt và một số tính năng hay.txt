Để thay đổi key binding:
	Ở cửa sổ ST4: Preferences -> [Key Bindings] -> sau khi thay đổi -> Ctrl+S

Một số phím tắt hay
	Ctrl+N: New file
	Ctrl+S: Save file
	Ctrl+W: Đóng file
	Ctrl+Z: Khôi phục(quay ngược thời gian)
	Ctrl+F: Tìm nội dung
	Ctrl+Shift+Backspace: Xóa một dòng
	Ctrl+Backspace: Xóa nhiều 
	Ctrl+Tab: di chuyển file từ trái sang phải
	Ctrl+Shift+Tab: di chuyển file từ phải sang trái
	Comment dòng nhanh: Ctrl+/
	Comment đoạn nhanh: Ctrl+Shift+/
	Ctrl+ mũi tên lên/xuống
Để có nhiều con trỏ ở nhiều chỗ: Ctrl+chuột trái vào nhiều chỗ khác nhau
Thay đổi tab size và syntaxx: góc phía dưới bên phải
