Ở cửa sổ ST4: 
Cách thay đổi color scheme:
	Ctrl+Shift+P -> "color"(UI: Select Color Scheme) -> Chọn một color scheme -> Enter

Cách làm nền trong suốt cho Command Prompt:
	Win+R -> "cmd" -> Ở cửa sổ cmd: ở thanh bên trên -> [Propeties] ->   Opacity -> thay đổi số %
Đổi font cho Command Prompt: ->[Font]
Đổi màu chữ và nền: [Colors] -> [Screen Text] -> [Screen Background]
